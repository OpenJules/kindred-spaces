

## Landing Page

A clean, modern landing page with smooth animations and responsive design.

### Sections
1. **Hero** — Bold headline, subtitle, and a primary CTA button with a subtle gradient background
2. **Features** — 3-column grid with icons highlighting key benefits
3. **How It Works** — 3-step process with numbered cards
4. **Testimonials** — Customer quotes in a card carousel
5. **CTA / Newsletter** — Final call-to-action with email signup
6. **Footer** — Links, social icons, and copyright

### Design
- Clean, minimal aesthetic with generous whitespace
- Smooth scroll-triggered fade-in animations
- Fully responsive (mobile, tablet, desktop)
- Consistent color palette using the existing design system
